# 安装指南

本指南将帮助你在不同的环境中安装和配置我们的产品。

## 系统要求

在开始安装之前，请确保你的系统满足以下要求：

- **Node.js**: 版本 18.0 或更高
- **包管理器**: pnpm、npm 或 yarn
- **操作系统**: Windows、macOS 或 Linux

## 安装步骤

### 使用 pnpm（推荐）

```bash
pnpm install
```

### 使用 npm

```bash
npm install
```

### 使用 yarn

```bash
yarn install
```

## 验证安装

安装完成后，运行以下命令验证安装是否成功：

```bash
pnpm docusaurus --version
```

如果看到版本号输出，说明安装成功。
